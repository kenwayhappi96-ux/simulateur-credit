## 📌 NOTES IMPORTANTES

### Questions Conditionnelles

Certaines questions ne s'affichent que selon les réponses précédentes :
- Questions sur le conjoint : uniquement si marié(e)/pacsé(e)/concubinage
- Questions sur le loyer : uniquement si locataire
- Questions sur le crédit immobilier : uniquement si propriétaire avec crédit
- Questions sur l'ancienneté professionnelle : sauf si retraité/sans emploi
- Questions sur les crédits : uniquement si crédits en cours
- Questions sur la fin de contrat : uniquement si CDD/Intérim

### Nombre de Questions par Parcours

- **Véhicule** : 16 questions de base + jusqu'à 8 questions conditionnelles = **24 questions max**
- **Travaux** : 13 questions de base + jusqu'à 2 questions conditionnelles = **15 questions max**
- **Équipement** : 11 questions de base + jusqu'à 2 questions conditionnelles = **13 questions max**
- **Argent** : 11 questions de base + jusqu'à 3 questions conditionnelles = **14 questions max**

### Validation des Montants

- **Revenus minimum** : 800 €/mois
- **Montant véhicule** : 1 000 € à 100 000 €
- **Montant travaux** : 1 000 € à 100 000 €
- **Montant équipement** : 500 € à 200 000 €
- **Montant argent** : 500 € à 75 000 €
